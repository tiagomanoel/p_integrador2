Sobre a Aplicação

Esta aplicação web foi desenvolvida para fornecer dados atualizados sobre cotações de moedas em Real Brasileiro (BRL), exibindo um breve histórico desses valores. O design da aplicação busca ser simples e objetivo, com uma performance ágil para garantir uma excelente experiência ao usuário.

Funcionalidades

  • Visualização em Tempo Real: As cotações são atualizadas em tempo real, garantindo que os usuários tenham acesso às informações mais recentes.
  • Histórico de Cotações: Os usuários podem visualizar um histórico das cotações, permitindo a análise de tendências ao longo do tempo.
  • Interface Intuitiva: O layout é projetado para ser simples e direto, facilitando a navegação e o acesso às informações.

Tecnologias Utilizadas

Django

Django é um framework web de alto nível em Python que promove o desenvolvimento rápido e um design limpo e pragmático. Vantagens incluem:

  • Estrutura sólida do projeto
  • Segurança eficiente
  • Escalabilidade

Django REST Framework

O DRF é uma biblioteca poderosa para a construção de APIs web com Django, permitindo:

  • Construção rápida de APIs
  • Serialização de dados
  • Autenticação e permissões

Hospedagem em Servidor Próprio

A aplicação está hospedada em um servidor próprio, utilizando Proxmox para virtualizar o mais recente Ubuntu Server. Vantagens incluem:

  • Flexibilidade na virtualização
  • Gerenciamento centralizado
  • Rápida recuperação de máquinas virtuais

Simplicidade e Usabilidade

A aplicação foi desenhada com um layout simples e limpo, proporcionando:

  • Foco no usuário
  • Responsividade
  • Facilidade de navegação

Em resumo, esta aplicação combina a robustez do Django, a flexibilidade do Django REST Framework e a eficiência da hospedagem em servidor próprio para fornecer uma solução escalável e de alto desempenho.
