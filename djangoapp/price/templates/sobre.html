Sobre a Aplicação

This web application was developed to provide updated data on currency quotes in Brazilian Real (BRL), showing a brief history of these values. The application's design aims to be simple and objective, with agile performance to ensure an excellent user experience.

Funcionalidades

  • Real-Time Visualization: Quotes are updated in real time, ensuring users have access to the latest information.
  • Quote History: Users can view a history of quotes, allowing for trend analysis over time.
  • Intuitive Interface: The layout is designed to be simple and direct, facilitating navigation and access to information.

Tecnologias Utilizadas

Django

Django is a high-level web framework in Python that promotes rapid development and clean, pragmatic design. Advantages include:

  • Solid Project Structure
  • Efficient Security
  • Scalability

Django REST Framework

The DRF is a powerful library for building web APIs with Django, allowing:

  • Fast API building
  • Data Serialization
  • Authentication and Permissions

Hospedagem em Servidor Próprio

The application is hosted on a private server, using Proxmox to virtualize the latest Ubuntu Server. Advantages include:

  • Flexibility in virtualization
  • Centralized management
  • Quick recovery of virtual machines

Simplicidade e Usabilidade

The application was designed with a simple and clean layout, providing:

  • User Focus
  • Responsiveness
  • Ease of Navigation

In summary, this application combines the robustness of Django, the flexibility of Django REST Framework, and the efficiency of hosting on a private server to provide a scalable and high-performance solution.
